# FontColor

## Introduction

This is a simple tool for you to check out font installed.

## Dependency

- C++ 20
- [Qt 6.9](https://www.qt.io) or later
- [Fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig)
- [SQLite](https://sqlite.org)
